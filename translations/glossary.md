# Hightouch 日本語化 — 用語カタログ

このカタログは、Hightouch UIの日本語翻訳における用語を統一するための公式リストです。
社内・クライアント間での表記ゆれを防ぎ、UI上・ドキュメント上・口頭での説明が一致するよう管理します。

---

## 用語統一ルール

1. **このカタログにある用語は必ずこの日本語訳を使う**（個人判断での変更不可）
2. 変更したい場合は GitHub の Issue または Pull Request で提案する
3. カタカナ表記は業界標準に合わせ、無理に漢字・ひらがなに変換しない
4. 固有名詞（製品名・サービス名）は翻訳しない

## カタログ管理ルール

### 追加・変更の提案方法

1. このリポジトリをフォークする
2. この `glossary.md` を編集する
3. Pull Requestを作成し、変更理由を記載する
4. DearOneチームがレビューして承認・マージ

### 翻訳しないもの

- **固有名詞**: Hightouch, Snowflake, BigQuery, Salesforce, HubSpot など製品名
- **ユーザーが入力したデータ**: カラム名、テーブル名、フィールド値
- **SQLコード・プログラムコード**
- **URL・メールアドレス**
- **数値・日付**（フォーマット変換は行わない）
