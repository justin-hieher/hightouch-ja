# 01. 概要

## この拡張機能とは

**Hightouch 日本語化**は、データ連携プラットフォーム [Hightouch](https://hightouch.com) のUIを日本語に翻訳するブラウザ拡張機能です。Chrome および Microsoft Edge に対応しています。

Hightouch は英語のみのUIで提供されており、日本語ユーザーにとって操作の理解に時間がかかる場合があります。この拡張機能をインストールするだけで、ナビゲーション・ボタン・ステータス表示などの主要なUI要素が自動的に日本語で表示されます。

---

## 何ができるか

| 機能 | 説明 |
|------|------|
| 自動翻訳 | `app.hightouch.com` を開くと自動的に日本語UIに切り替わる |
| ON/OFF切り替え | 拡張機能アイコンをクリックして、いつでも英語UIに戻せる |
| 用語の統一 | 社内・クライアント間で一貫した日本語訳を適用 |
| データ保護 | カラム名・セル値・SQLコードなどのユーザーデータは翻訳しない |
| カスタマイズ | 辞書ファイルを編集して独自の翻訳を追加できる |

---

## 翻訳される主な画面

- **ナビゲーション**: 同期・モデル・送信先・ソース・オーディエンス・ジャーニー・設定 など
- **ステータス表示**: 実行中・完了・失敗・待機中 など
- **アクションボタン**: 作成・保存・編集・削除・接続 など
- **メッセージ**: エラーメッセージ・確認ダイアログ・通知 など

---

## 翻訳されないもの（重要）

以下は**意図的に翻訳対象外**としています：

- **ユーザーが作成したデータ**: テーブル名・カラム名・フィールド値・プレビューデータ
- **SQLエディタ内のコード**
- **固有名詞（製品名・サービス名）**: Snowflake、BigQuery、Salesforce、HubSpot など
- **数値・URL・メールアドレス**
