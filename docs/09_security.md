# セキュリティ仕様書

**対象読者**: 導入を検討する企業の IT 担当者・情報セキュリティ部門

---

## 概要

Hightouch 日本語化拡張機能は、Hightouch の UI テキストをブラウザ内でのみ日本語に置換します。
**外部サーバーへのデータ送信は一切行いません。**

---

## パーミッション（権限）の説明

Chrome/Edge の拡張機能はインストール時に権限を要求します。
本拡張機能が要求する権限は以下の 2 つのみです。

| 権限 | 用途 | 取得するデータ |
|------|------|--------------|
| `storage` | ON/OFF の設定を保存する | オン・オフの状態（true/false）のみ |
| `host_permissions`: app.hightouch.com | Hightouch のページで動作する | なし（ページへのアクセス権のみ） |

### 要求していない権限（できないこと）

以下の権限は**要求していない**ため、技術的に不可能です。

| 権限 | できないこと |
|------|------------|
| `tabs` | 閲覧中の URL 一覧・タブ履歴の取得 |
| `cookies` | Cookie・セッショントークンの読み取り |
| `webRequest` | 通信内容の傍受・改ざん |
| `history` | 閲覧履歴の取得 |
| `downloads` | ファイルのダウンロード |

## データの取り扱い

### 読み取るデータ

拡張機能は `https://app.hightouch.com` のページ上に表示されているテキストを読み取り、
英語→日本語の置換を**ブラウザ内のみで**行います。

### 送信するデータ

**外部への送信は一切ありません。**

本拡張機能が行うネットワーク通信は以下のみです：

- 拡張機能に同梱された翻訳辞書（`translations/ja.json`）の読み込み
  → インターネットへの通信ではなく、ブラウザ内の読み込みです

### 保存するデータ

`chrome.storage.sync` に以下のみ保存します：

```json
{ "enabled": true }
```

翻訳のオン・オフ状態を示すフラグのみです。Hightouch のデータや個人情報は保存しません。

---

## 翻訳対象から除外しているデータ

Hightouch に表示されるユーザーデータ（SQL クエリ・テーブルのセル値・入力フォームの内容等）は
翻訳の対象から**明示的に除外**しています。

除外対象の例：

- テキスト入力欄・テキストエリア（`input`, `textarea`）
- SQL エディタ（CodeMirror / Monaco Editor）
- データプレビューのセル値（`table td`, `[role="gridcell"]`）
- コードブロック（`code`, `pre`）

これらの要素は読み取りも置換も行いません。

また、**400文字を超えるテキストは翻訳対象から除外**しています。UIのラベル・ボタン・説明文は通常400文字未満であるため、それを超える長いテキストはユーザーデータと判断してスキップします。これにより、除外セレクタをすり抜けた場合でもユーザーデータが `data-ht-original` 属性に誤って保存されることを防いでいます。

---

## DOM への変更について

翻訳の実装上、ページの HTML に以下の変更を加えます：

- UI テキストノードの内容を日本語に書き換える
- 翻訳済み要素に `data-ht-original` という属性を付与する（元の英語テキストを保持）

`data-ht-original` 属性は復元・デバッグ目的で使用するもので、外部に送信されることはありません。
拡張機能を OFF にすると、すべての変更が元に戻ります。

## Content Security Policy（コンテンツセキュリティポリシー）

`manifest.json` に以下の CSP を明示しています：

```json
"content_security_policy": {
  "extension_pages": "script-src 'self'; object-src 'self';"
}
```

これにより、技術的に以下が禁止されます：

- 外部サーバーからのスクリプト読み込み
- `eval()` などの動的コード実行
- 外部オブジェクト（Flash 等）の埋め込み

---

## コードの監査可能性

本拡張機能は OSS（オープンソース）として GitHub で公開しています。

- ソースコードはすべて公開されており、社内のエンジニアが確認できます
- 難読化・minify は行っていません（読めるコードをそのまま配布）
- `translations/ja.json` の翻訳内容もすべて確認可能です

## 配布と更新

### 配布方法

Chrome Web Store ではなく、GitHub Releases から zip ファイルとして配布しています。

**メリット**: Chrome の自動アップデートが適用されないため、意図しないコード変更が加わりません。

**注意点**: セキュリティアップデートがあった場合は手動での更新が必要です。
更新手順は [08_upgrade.md](08_upgrade.md) を参照してください。

### 配布ファイルの整合性確認（SHA256）

各リリースの GitHub Releases ページには SHA256 チェックサムを掲載しています。
ダウンロードした zip ファイルが改ざんされていないか、以下のコマンドで確認できます：

**Mac / Linux:**
```bash
shasum -a 256 hightouch-ja-v1.0.0.zip
```

**Windows（PowerShell）:**
```powershell
Get-FileHash hightouch-ja-v1.0.0.zip -Algorithm SHA256
```

表示された値が Releases ページの値と一致すれば、正規のファイルです。

## よくある質問

**Q. Hightouch に入力したデータ（顧客データ等）は拡張機能に読み取られますか？**

A. データプレビューのセル値・入力フォームは翻訳対象から除外しており、読み取りも置換も行いません。ただし、UI上に表示されているテキスト（ナビゲーション・ボタン・ラベル等）は翻訳のために読み取ります。いずれも外部への送信は行いません。

**Q. 拡張機能が通信しているか確認する方法はありますか？**

A. Chrome DevTools の「ネットワーク」タブを開いた状態で Hightouch を操作することで、通信内容をリアルタイムに確認できます。拡張機能による外部通信は発生しません。

**Q. 拡張機能を削除したらデータは消えますか？**

A. はい。`chrome.storage.sync` に保存されているオン・オフフラグは拡張機能の削除と同時に消去されます。
