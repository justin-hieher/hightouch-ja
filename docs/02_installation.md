# 02. インストール手順

拡張機能ファイルをダウンロードしてブラウザに読み込む手順を説明します。
Chrome と Microsoft Edge のどちらでも同じ手順でインストールできます。

---

## 事前準備：拡張機能ファイルのダウンロード

1. [GitHub リリースページ](https://github.com/justin-hieher/hightouch-ja/releases) を開く
2. 最新バージョンの `hightouch-ja-vX.X.X.zip` をクリックしてダウンロード
3. ダウンロードした `.zip` ファイルを展開（解凍）する

> 📁 展開後に `hightouch-ja/` というフォルダが作成されます。このフォルダを任意の場所に保存してください（削除するとインストールが解除されます）。

---

## Chrome へのインストール

### ステップ 1：拡張機能管理ページを開く

アドレスバーに以下を入力して Enter を押します：

```
chrome://extensions
```

または、右上の「⋮」メニュー → **その他のツール** → **拡張機能** をクリックします。

### ステップ 2：デベロッパーモードを有効にする

ページ右上のトグル「**デベロッパー モード**」を **ON** にします。

> ⚠️ デベロッパーモードをONにしないと、次のステップが表示されません。

### ステップ 3：フォルダを読み込む

「**パッケージ化されていない拡張機能を読み込む**」ボタンをクリックします。

フォルダ選択ダイアログが開くので、先ほど展開した **`hightouch-ja` フォルダ**を選択して「開く」をクリックします。

### ステップ 4：インストール完了の確認

拡張機能一覧に「**Hightouch 日本語化**」が表示されれば成功です。

---

## Microsoft Edge へのインストール

Edge も Chrome と同じ手順でインストールできます。

### ステップ 1：拡張機能管理ページを開く

アドレスバーに以下を入力して Enter を押します：

```
edge://extensions
```

または、右上の「…」メニュー → **拡張機能** → **拡張機能の管理** をクリックします。

### ステップ 2：デベロッパーモードを有効にする

左サイドバー下部の「**開発者モード**」トグルを **ON** にします。

### ステップ 3：フォルダを読み込む

「**展開して読み込む**」ボタンをクリックし、`hightouch-ja` フォルダを選択します。

### ステップ 4：インストール完了の確認

拡張機能一覧に「**Hightouch 日本語化**」が表示されれば成功です。

---

## 動作確認

1. ブラウザで `https://app.hightouch.com` を開きます
2. ログイン後、ナビゲーションメニューが日本語（「同期」「モデル」「送信先」など）で表示されていることを確認します
3. ブラウザ右上に拡張機能のアイコンが表示されていることを確認します（ない場合はパズルのピースアイコン → 📌ピン留めで表示できます）

---

## アンインストール

1. `chrome://extensions`（または `edge://extensions`）を開く
2. 「Hightouch 日本語化」の「削除」ボタンをクリック

---

## よくある質問

**Q: 「デベロッパーモード」は危険ですか？**
A: デベロッパーモードはサードパーティ製の拡張機能（Chrome Web Store 未公開）をインストールするための設定です。この拡張機能はオープンソースでコードが公開されているため、安全性を自分で確認することもできます。

**Q: インストール後にフォルダを移動・削除してしまいました**
A: 拡張機能が無効になります。再度インストール手順を最初からやり直してください。

**Q: 翻訳されない部分があります**
A: [07. トラブルシューティング](./07_troubleshooting.md) を参照してください。
